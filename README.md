# 🌆 AI-Powered City Guide Agent

## 📌 Overview
An AI-powered agent that provides quick and reliable city information through a single query.

## 🚀 Features
- Accepts city name via API  
- Generates instant AI-based responses  
- Integrates Wikipedia for data  
- Uses Gemini for response generation  
- Deployed on Cloud Run  

## 🏗️ Tech Stack
- Python, FastAPI  
- Google ADK  
- Gemini (LLM)  
- Cloud Run  
- Wikipedia API  

## ⚙️ Workflow
User Input → ADK Agent → Wikipedia Tool → Gemini Response → API Output  

## ☁️ Deployment
Deployed on Google Cloud Run with a public HTTP endpoint.  

## 💡 Use Case
Can be extended into a travel assistant (hotels, weather, trip planning).
